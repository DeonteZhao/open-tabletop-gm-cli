# System Module — Call of Cthulhu 7th Edition (CoC 7e)

---

## Dice Convention

**核心机制 (Core resolution):** 
所有检定均使用百分骰（D100）。玩家掷出的数值需要 **小于或等于** 对应的属性或技能值即为成功。

**成功等级 (Success thresholds):** 
- **大成功 (Critical Success):** 掷出 01。
- **极难成功 (Extreme Success):** 掷出值 ≤ 技能或属性值的五分之一（1/5）。
- **困难成功 (Hard Success):** 掷出值 ≤ 技能或属性值的二分之一（1/2）。
- **常规成功 (Regular Success):** 掷出值 ≤ 技能或属性值。
- **失败 (Failure):** 掷出值 > 技能或属性值。
- **大失败 (Fumble):** 若技能值 < 50，掷出 96-100 为大失败；若技能值 ≥ 50，掷出 100 为大失败。

**奖励骰/惩罚骰 (Bonus/Penalty Dice):**
等同于优势/劣势机制。投掷一次个位骰，但投掷多次十位骰（通常增加1颗）。
- **奖励骰:** 取十位骰中较**小**的数值组合。
- **惩罚骰:** 取十位骰中较**大**的数值组合。

**孤注一掷 (Pushing the Roll):**
当常规检定失败时，玩家可以描述角色如何采取进一步行动来重新尝试检定。若“孤注一掷”失败，通常会带来更严重的灾难性后果。（注意：理智检定、战斗检定、伤害检定、幸运检定等不能孤注一掷）。

**战斗范例:**
`[调查员] 攻击 [邪教徒]: [斗殴:50] 掷出 = [24] (困难成功) vs [闪避:30] 掷出 = [45] (失败) — 攻击命中! [伤害掷骰] = [1D3] + [1D4 DB] = [4] 点物理伤害`

## Ability Scores / Statistics

属性值范围通常在 15-90 之间。

| 属性 (Stat) | 缩写 | 描述 (Description) |
|-------------|------|--------------------|
| 力量 (Strength) | STR | 决定角色的肌肉力量、近战伤害加值 (DB) 和体格 (Build)。 |
| 体质 (Constitution) | CON | 决定生命值、抗毒、抗病能力。 |
| 体型 (Size) | SIZ | 决定身高体重、生命值、DB 和体格。 |
| 敏捷 (Dexterity) | DEX | 决定角色的速度、闪避能力和战斗先攻。 |
| 外貌 (Appearance) | APP | 决定角色的外在魅力和第一印象。 |
| 智力 (Intelligence) | INT/Idea | 决定角色的学习能力、分析能力和“灵感”检定。 |
| 意志 (Power) | POW | 决定魔法值 (MP)、理智值上限、以及抵抗魔法和精神攻击的能力。 |
| 教育 (Education) | EDU/Know | 决定角色的常识积累和职业技能点数。 |

**衍生属性:**
- **生命值 (Hit Points, HP):** (CON + SIZ) / 10（向下取整）。
- **魔法值 (Magic Points, MP):** POW / 5。
- **伤害加值 (Damage Bonus, DB) & 体格 (Build):** 由 STR + SIZ 的总和决定。
- **移动力 (Move Rate, MOV):** 基于 STR, DEX, 和 SIZ 的比较（通常为 7, 8, 或 9）。

## Character Structure

| 字段 (Field) | 备注 (Notes) |
|-------------|-------------|
| 生命值 (HP) | 承受物理伤害时减少，降至0会导致濒死或死亡。 |
| 理智值 (Sanity, SAN) | 遇到恐怖事物时减少。当前理智值为0代表永久疯狂。 |
| 幸运 (Luck) | 0-99，可用于幸运检定，或花费幸运值改变其他检定的结果（如果开启该可选规则）。 |
| 魔法值 (MP) | 施放法术或启动某些魔法物品时消耗。 |
| 护甲 (Armor) | 减轻受到的物理伤害数值。 |
| 技能成长 (Skill Checks) | 标记本次游戏中成功使用过的技能，用于幕间成长。 |

## Health and Damage

**伤害机制:**
- **常规伤害:** 直接扣除 HP。
- **重伤 (Major Wound):** 如果单次受到的伤害 ≥ 最大 HP 的一半，角色将进入“重伤”状态。重伤状态下倒地，并且需要立即进行一次 CON 检定，失败则失去意识。

**恢复:**
- **急救/医学:** 成功使用急救回复 1D3 HP；成功使用医学回复 1D3 HP。
- **自然恢复:** 
  - 未受重伤：每天恢复 1 HP。
  - 重伤状态：每周进行一次 CON 检定，成功则恢复 1D3 HP 或消除重伤状态（具体取决于掷骰结果，极难成功可两者兼得）。

## Primary Resource

在 CoC 7e 中，资源管理主要集中在以下三项：

**1. 理智值 (Sanity, SAN)**
- **范围:** 0-99（上限为 99 - 克苏鲁神话技能值）。
- **消耗:** 遭遇超自然现象或恐怖事件时，进行理智检定 (1D100 vs 当前 SAN)。成功扣除较少（或不扣），失败扣除较多。
- **恢复:** 击败怪物、完成模组目标、接受心理治疗等。

**2. 幸运 (Luck)**
- **范围:** 0-99。
- **消耗:** 进行幸运检定（通常不扣除），或（如果开启花费幸运可选规则）可以 1:1 的比例花费幸运点数来降低失败的掷骰结果使其成功。
- **恢复:** 每次模组结束后进行幸运恢复检定。

**3. 魔法值 (Magic Points, MP)**
- **范围:** 0 至 (POW/5)。
- **消耗:** 施法、对抗魔法。
- **恢复:** 每天自然恢复 1 点，或通过特定法术、物品恢复。MP 降至 0 时，进一步消耗的 MP 从 HP 中扣除。

## Rests and Recovery

- **短期休息 (短时休整):** 无特殊系统机制，主要用于包扎伤口（急救）、恢复体力。
- **长期休息 (自然恢复):** 每天恢复 1 HP。MP 每天恢复 1 点。
- **精神恢复 (心理治疗):** 每月一次精神分析检定，成功可恢复 1D3 点 SAN。在家修养可进行自我恢复。

## Incapacitation and Death

- **失去意识 (Unconscious):** 当 HP 降至 0 但未受重伤，或者受到重伤且 CON 检定失败时，角色失去意识。
- **濒死 (Dying):** 当 HP 降至 0 且受到重伤时，角色进入濒死状态。在每个回合结束时，必须进行一次 CON 检定，失败则死亡。只有成功接受“急救”才能稳定伤势脱离濒死状态。
- **即死 (Instant Death):** 单次受到的伤害 ≥ 角色最大 HP，角色当场死亡。
- **疯狂 (Insanity):**
  - **临时疯狂 (Temporary Insanity):** 单次失去 ≥ 5点 SAN，并进行一次智力 (Idea) 检定，若成功理解了恐怖的本质，则进入临时疯狂（持续 1D10 小时）。
  - **不定性疯狂 (Indefinite Insanity):** 在一个游戏日（游戏内时间）内累积失去相当于当前 SAN 值五分之一 (1/5) 的理智值，角色进入不定性疯狂（持续直到被治愈）。
  - **永久疯狂 (Permanent Insanity):** SAN 降至 0。调查员不再可控。

| 状态 (Condition) | 严重程度 (Severity) | 效果说明 (Effect summary) |
|-----------------|---------------------|-------------------------|
| 濒死 (Dying) | critical | HP为0且有重伤，每回合需CON检定，失败即死。 |

## Advancement

**调查员成长阶段 (Investigator Development Phase):**
- 在游戏过程中，每当调查员成功使用了一个技能（必须是在有压力的情况下），该技能名称旁会打上一个标记（打勾）。
- 在模组结束或到达特定的成长阶段时，对每个被标记的技能进行一次 **成长检定**。
- **成长检定机制:** 掷 D100。如果结果 **大于** 当前技能值，或者大于 95，则技能获得成长。
- **成长数值:** 掷 1D10，将其加到该技能值上。
- **注意:** 每次成长阶段结束后，清除所有技能的标记。

## Bold Play Reward

**奖励名称:** 幸运点奖励 / 孤注一掷 (Pushing the Roll)

**效果:** 
1. **孤注一掷:** 允许玩家重掷失败的技能检定，但必须提出合理的理由，且失败会面临更严重的后果。
2. **花费幸运 (可选规则):** 允许玩家消耗幸运值以 1:1 的比例降低掷骰结果，使其达到成功阈值（不可用于理智、伤害、幸运或战斗检定）。

**如何获得:** 
守秘人 (KP) 可以在玩家有出色的角色扮演、提出绝妙的计划或做出了牺牲时，奖励 1D10 点幸运值。

## Campaign Arc Preferences

**首选战役模式 (Preferred campaign mode):** imported (导入模组) 或 improvised (即兴生成)。CoC以导入经典官方模组居多。
**典型剧情结构 (Typical arc structure):** structured — 通常是 linear (线性调查) 或 hub-and-spoke (网状调查，如阿卡姆城内的自由探索)。
**类型约定 (Genre conventions):** 
- **洛夫克拉夫特式恐怖:** 强调人类在宇宙中的渺小，调查越深入，面临的危险和理智损失越大。
- **节奏 (Pacing):** 通常由“调查期（线索收集、解密）”逐渐过渡到“对抗期（面对神话生物或邪教徒，生存压力剧增）”，最终达到高潮。战斗往往是致命的，逃跑或寻找替代方案通常比正面对抗更明智。

## Additional System Notes

- **追逐规则 (Chase Rules):** 
  当发生追逐时，角色通过 CON 检定决定移动力 (MOV) 的增减。使用行动点 (Action Points) 进行移动、攻击或克服障碍（使用各种技能如攀爬、跳跃等）。
- **战斗动作:** 
  - **反击 (Fight Back):** 受到近战攻击时，可选择反击。双方进行战斗技能对抗，成功等级高的一方造成伤害。平局时攻击方胜出。
  - **闪避 (Dodge):** 受到近战攻击时，可选择闪避。成功等级高于攻击方即可避开伤害。平局时闪避方胜出。
  - **战技 (Maneuver):** 用于缴械、推倒、擒抱等。需要比较双方体格 (Build)，体格差距过大会带来惩罚骰或无法进行战技。
